# 斗鱼弹幕飘屏 🎣💬

在 Windows 桌面上显示斗鱼实时弹幕的透明置顶工具，适合全屏游戏、直播监看、录屏等场景。

---

## ✨ 功能亮点

- 🔌 **快速连接房间**：支持输入房间号，或直接粘贴斗鱼直播间链接自动解析。
- 🪟 **透明置顶飘屏**：弹幕在独立窗口滚动显示，尽量不遮挡主画面。
- 🖱️ **鼠标穿透**：开启后可直接操作下层应用，不会误点到弹幕窗口。
- 🧭 **区域可控**：支持全屏、半屏（上下左右）、四分之一屏（四角）。
- 🖥️ **多显示器**：可在设置中选择飘屏落在哪一块屏幕上（默认主显示器）。
- 🔄 **更新检测**：可检查 GitHub 上是否有新版本并前往下载页。
- 🎨 **样式可调**：字号、透明度、速度、轨道间距、统一颜色都可配置。
- 🧹 **过滤能力**：支持屏蔽词、昵称黑名单、机器人弹幕过滤。
- 🔁 **重复文案策略**：可全量显示、去重，或合并为连击样式。
- 🧪 **模拟弹幕调试**：无需真实房间也能预览布局和效果。
- 💾 **本地配置持久化**：关闭主窗口后可驻留系统托盘继续运行。

---

## 🖥️ 系统要求

- 操作系统：**Windows 10/11**
- Node.js：建议 **18+**
- 包管理器：**pnpm**

---

## 🚀 快速开始（开发）

```bash
pnpm install
pnpm dev
```

常用命令：

```bash
pnpm typecheck
pnpm build
pnpm dist          # 打包 Windows 便携版（输出到 release/）
pnpm dist:dir      # 仅解包目录（调试目录结构）
```

发布到 GitHub Release 时，**上传便携版 exe**（文件名以 `release/` 下产物为准），无需附带 `latest.yml`、`*.blockmap`、`builder-debug.yml` 等构建辅助文件。

---

## 📦 安装与使用（最终用户）

1. 前往 [Releases](https://github.com/qianjiachun/douyu-danmaku-overlay/releases)，下载当前版本的 **便携版 exe**（文件名以 Assets 列表为准，无需安装向导）。
2. 可将 exe 放在任意文件夹中运行；**配置与缓存**默认保存在 **与 exe 同目录下的 `data` 文件夹** 内。若同目录已有名为 `data` 且不是文件夹（例如同名文件），程序会自动改用 **`data.douyu-danmaku-overlay`** 目录。
3. 启动后在主界面填写房间号并开启飘屏。
4. 如遇无弹幕，先确认直播间本身有弹幕，再尝试「重连弹幕」。

---

## ⚙️ 设置项说明

| 项目 | 说明 |
|---|---|
| 飘屏显示器 | 选择弹幕覆盖的屏幕；默认跟随系统主显示器 |
| 飘屏区域 | 控制弹幕窗口显示范围（全屏/半屏/四分之一屏，相对所选显示器） |
| 鼠标穿透 | 点击直接穿透到下层应用 |
| 字号/透明度/速度 | 调整可读性与动态效果 |
| 同屏上限/队列上限 | 控制密度与性能占用 |
| 屏蔽词/昵称黑名单 | 过滤不想看到的内容 |
| 重复弹幕模式 | 全显、去重、连击合并 |
| 斗鱼颜色 | 使用消息原始颜色或统一颜色 |
| 模拟弹幕 | 本地调试显示效果 |

> 修改配置后请保存并应用，具体交互以界面为准。

---

## 🧰 托盘行为

- 关闭主窗口后，程序默认可继续驻留托盘。
- 飘屏开启状态下，弹幕会持续显示。
- 需要完全退出时，请使用托盘菜单中的退出选项。

---

## ❓ 常见问题

**Q: 显示已连接，但没有弹幕？**  
A: 先确认房间确实有实时弹幕，再执行“重连弹幕”或重新保存配置。

**Q: 飘屏影响我点击其他窗口？**  
A: 开启“鼠标穿透”或缩小飘屏区域即可。

**Q: 是否支持 macOS/Linux？**  
A: 当前版本以 Windows 为主，其他平台暂未承诺支持。
