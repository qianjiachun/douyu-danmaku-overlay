# 斗鱼弹幕飘屏

Windows 桌面端辅助工具：在透明置顶窗口中展示斗鱼直播间弹幕，适用于全屏游戏、全屏播放等场景。

</div>

---

## 简介

本软件在 **Windows** 环境下运行，通过独立飘屏窗口展示指定斗鱼房间的实时弹幕。窗口可设置为透明、置顶，并支持限定显示区域与鼠标穿透，以降低对前台操作的干扰。

---

## 主要特性

**连接与房间**

- 支持通过 **房间号** 或 **斗鱼房间页链接** 解析并进入指定直播间。
- 设置页提供 **连接状态** 展示；支持在保留当前配置的前提下 **手动重连** 弹幕源。

**显示与布局**

- **透明置顶** 飘屏，弹幕自一侧横向滚动展示。
- **飘屏区域** 可配置为全屏、半屏（上/下/左/右）或四分之一屏（四角之一），默认以 **主显示器** 为参照。
- 支持 **鼠标穿透**：启用后，鼠标事件作用于下层应用，而非飘屏窗口。

**外观与性能相关参数**

- 可配置 **字号**、**统一文字颜色**、**窗口整体透明度**、**滚动速度**。
- 可选使用 **斗鱼消息中的颜色字段** 展示弹幕颜色，否则采用上述统一颜色。
- 可限制 **同屏最大条数** 与 **队列最大长度**，用于控制展示密度与资源占用。
- 可配置 **轨道间距**（弹幕行之间的留白）。

**内容与过滤**

- **屏蔽词列表**：命中规则的内容不予展示。
- **发送者昵称黑名单**（子串匹配、不区分大小写）：匹配则整条不展示。
- **重复文案策略**：支持每条均展示、同屏与队列中去重仅保留一条，或 **合并为连击样式**（可配置合并时间窗口）。
- 可选 **过滤机器人弹幕**（依据协议字段特征，减少无意义刷屏）。

**调试与维护**

- 支持 **模拟弹幕**：在不依赖真实直播间的情况下验证显示效果（需同时开启飘屏）。
- 配置在本地 **持久化保存**；主窗口关闭后程序 **驻留系统托盘**，可通过托盘菜单快速开关飘屏。

## 使用说明

1. 在主界面填写 **房间号** ，点击 **开启飘屏**。
2. 观察 **连接状态**：在已连接服务器且已登录房间后，弹幕应在飘屏区域内滚动显示。
3. 若连接异常或长时间无弹幕，可在确认直播间本身有弹幕的前提下，尝试 **重新连接弹幕** 或再次保存配置。

## 设置项概要

| 类别 | 说明 |
|------|------|
| 飘屏区域 | 限定弹幕窗口在主显示器上的几何范围（全屏/半屏/四分之一屏）。 |
| 鼠标穿透 | 是否将鼠标操作穿透至下层窗口。 |
| 字号 / 颜色 / 透明度 / 速度 | 控制可读性与视觉强度及滚动快慢。 |
| 屏蔽词、昵称黑名单 | 按规则过滤弹幕内容或发送者。 |
| 重复弹幕模式 | 控制相同文案的展示与合并行为。 |
| 斗鱼颜色 | 是否采用消息中的颜色信息，而非单一前景色。 |
| 过滤机器人弹幕 | 是否依据协议特征丢弃疑似机器人消息。 |
| 模拟弹幕 | 用于本地验证布局与样式，与真实房间无关。 |

各项参数调整后，需 **保存并应用** 方可生效（具体交互以软件界面为准）。

---

## 托盘与退出

关闭主窗口后，程序通常仍 **驻留托盘**，在 **开启飘屏** 的前提下弹幕会继续显示。完全退出请通过托盘图标的 **右键菜单** 选择退出或关闭（菜单文案以实际版本为准）。

## 常见问题

**已显示连接成功，但无弹幕。**  
请先确认该直播间当前确有弹幕；可尝试重新连接或重新保存房间配置。

**飘屏妨碍点击下层界面。**  
请启用 **鼠标穿透**，或缩小 **飘屏区域**。

**是否支持 macOS、Linux 或移动设备。**  
当前发布以 **Windows** 为主；其他平台不在本说明承诺范围内。
